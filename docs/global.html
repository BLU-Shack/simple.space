ClientOptions

The options supplied on instantiating the client.
Type:
  • object
Properties:
Name Type Attributes Default Description
botToken string <optional>
 The bot's API Token (not the bot's Discord token!) Required for most endpoints.
botID string <optional>
 The bot's ID.
userToken string <optional>
 Your site token. Required for most endpoints.
cache boolean <optional>
 false Whether or not to cache bots, guilds, emojis, or users on fetch.
autoCache boolean <optional>
 false Whether or not to automatically cache all bots.
autoCacheInterval number <optional>
 60000 The number of milliseconds to wait for each auto cache interval.
version number <optional>
 1 The default version of the API to use when fetching and posting.
